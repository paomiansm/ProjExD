# 第 6 回
## 工科大を守ろう！こうかとん（ ex06/ProtecttheTUT.py)

### ゲーム背景
- ある夜に、東京工科大学の上空に突然たくさんのUFOが出現し、大学に侵攻しょうとする。守護神である
こうかとんが学校を守るために敵と戦う。

### ゲーム概要
- ex06/ProtecttheTUT.pyを実行すると，1366x768のスクリーンに夜の工科大の上空が描画され，こうかとんが画面の下に描画される。
- こうかとんの真っ正面に敵がいっぱい出現し、こうかとんの方向に近づく
- 画面の下でこうかとんを移動させ、敵を消滅する。
- 敵が一つを消滅したら、ランダム一つの敵が復活する
- 敵がこうかとんにぶつかったらゲーム終了

### 操作方法
- “←”が左に移動する。
- “→”が右に移動する
- “SPACE”で銃弾を打つ
